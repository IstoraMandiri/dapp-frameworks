# Truffle Example

### A quest to find the perfect dapp development environment

## Install

* Truffle `npm install -g truffle`
* Imba `npm install -g imba`

## Usage

Spin up a testnet with `embark` on port `8545`, then start watching with `truffle`

```bash
truffle deploy
truffle build
truffle watch
```

Run the tests

```
truffle test
```
